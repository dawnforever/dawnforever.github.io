使用油猴插件刷课

发表于 2023-04-19 更新于 2023-04-20 分类于 刷课 阅读次数： Valine：

智慧树

下载油猴插件

推荐火狐浏览器，edge也可

  • 油猴官网
  • 点击安装，如图：

    安装完成后，点击扩展图标，点击“眼睛”图标，使油猴显示在工具栏。

搜索脚本

点击油猴插件，点击“获取新脚本”，搜索“智慧树”，选择第一个“OCS网课助手”，点击安装。（懒人链接OCS网课助手）


开始刷课

  • 安装完成打开智慧树，打开你要刷的课程，插件就能自动开始刷课了，包括自动答题，上个视频刷完自动刷下个，完全不用你管，还可设置刷课时间，效果如下：
  • 该插件还适用于其他平台，可自行探索。

国家中小学智慧教育平台

脚本安装

该脚本是为了打破平台视频不能倍速，不可拖动进度条的限制

同样是基于油猴插件，这次的插件可以提高视频倍速，最高是16x，也可慢速播放，由于本次的脚本搜不到，所以这里直接放链接:HTML5视频播放器增强脚本

操作方法

安装完成后，就可以开始操作了，方法很简单，打开视频后，按 C 提高倍速，按 X 是降低倍速，按 Z 是恢复初始倍速，如果16X 还嫌慢那就直接 CTRL ＋ ← 或 → ，拖动进度条，值得注意的是，该插件并不适合智慧树这种带有平时分的网课平台，且在“国家中小学智慧教育平台的教师研修模块”使用时，要注意视频的结尾有一道题，拖进度条的时候注意停一下，答题，否则可能不会记录学时。
